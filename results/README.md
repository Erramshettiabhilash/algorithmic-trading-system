# Results

Generated outputs belong here:

- Charts
- Calibration reports
- Simulation summaries
- Stress-test outputs
- Hedging P&L diagnostics

Most generated files in this folder are ignored by Git so the repository stays clean.

The `examples/` subfolder is intentionally kept as a small reproducible showcase
generated by:

```powershell
.\.venv\Scripts\python.exe -m examples.run_full_demo
```
