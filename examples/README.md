# Examples

This folder contains reproducible scripts that exercise the platform.

Run the original options analytics demo from the project root:

```powershell
.\.venv\Scripts\python.exe -m examples.run_full_demo
```

Run the AI quant research demo:

```powershell
.\.venv\Scripts\python.exe -m examples.run_ai_quant_demo
```

The scripts write charts and summary files to:

```text
results/examples/
```

The AI quant demo writes to:

```text
results/examples/ai_quant_demo/
```
